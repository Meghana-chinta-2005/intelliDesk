# IntelliDesk

IntelliDesk is a RAG-based AI support ticket resolver for internal employee support questions.

This repository is being built in phases:

1. Project scaffolding
2. Knowledge base
3. Ingestion and chunking
4. Embeddings and vector store
5. Retrieval engine
6. Generation layer
7. Evaluation
8. Backend API
9. Streamlit frontend
10. Testing
11. Docker
12. CI/CD
13. Documentation

## Current Status

Phase 1 scaffold is in place. Implementation details will be added in later phases.
